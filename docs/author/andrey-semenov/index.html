How to use git patch

 Mon 21 June 2021  —  Andrey Semenov

Creating the patch

git format-patch -1 <sha> OR git format-patch -1 HEAD

Applying the patch

git apply --stat file.patch # show stats.
git apply --check file.patch # check for error before applying
git am < file.patch # apply the patch finally

Source: https://stackoverflow.com/questions/6658313/generate-a-git-patch-for-a-specific-commit
